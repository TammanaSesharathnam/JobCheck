# JobCheck Frontend (Client)

This is the React + Tailwind CSS frontend for JobCheck.

## Prerequisites
- Node.js installed.

## Setup
1. Navigate to the client directory:
   ```sh
   cd JobCheck/client
   ```
2. Install dependencies (if not already done):
   ```sh
   npm install
   ```

## Running the App
Start the development server:
```sh
npm run dev
```

Open your browser at the URL shown (usually `http://localhost:5173`).

## Features
- **Login Page**: Dark/Gold theme, role selection.
- **Register Page**: Detailed form with role selection.
- **Responsive Design**: Works on mobile and desktop.
